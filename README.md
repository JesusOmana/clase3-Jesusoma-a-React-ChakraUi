    Pagina de producto de Suculenta

En esta pagina se presenta un producto en este caso la suculenta Echeveria elegans;
donde tiene una imagen, descripcion, si el tipo de planta es para ambiente soleado,
el stock disponible, Sku un identificador unico de producto, el precio del producto
y un boton de compra.

El codigo usa la informacion proporcionada de los porductos para aplicarlos al funcionamiento
y a su ves pueda ser responsive a cualquier dispositivo, identificando cada uno y ubicandolos
automaticamente en la pagina, determina si el producto es de ambiente soleado, si hay stock disponible
y de lo contrario el boton de compra se desactiva.

Se usaron 2 tecnologia; React Vite + Chakra Ui (SE APLICO EL DEPLOY)
